# exoplanet-terminal
Live Demo: https://replit.com/@PoojithBestha/exoplanet-terminal

I built this text-based terminal using Python concepts I learned in my 12th-grade computer science class to challenge myself for the FlavorTown Space Sidequest. It uses the csv module to read a dataset of exoplanets and allows the user to search by name, filter for habitable worlds, and calculate the average mass of the planets.
